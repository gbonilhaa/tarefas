após inicializar a vm

1- use o "pwd" para mostrar onde vc esta

2- usando o "cd" va ate a pasta do vagrant e execute os comandos..

3- usar o comando "rails new nome_projeto" para criar um novo projeto use o -B para pular o processo "bundle" ( comando que faz o donwload e instalacao automatica de TODOS os arquivos,dados, drivers etc "dependencias" que o projeto precisa para funcionar.)

4- depois que criado o projeto, use o comando "cd nome_projeto" para entrar na pasta do projeto.

5- inicializar um projeto git
	git e um comando que faz o gerenciamento do projeto criado, com ele sao feitas as alteracoes no projeto.
	("é ele que gerencia tudo, e o github apresenta de uma forma mais bonita para o usuario/programador")

6- adicionar todos os arquivos ao git (todo o projeto sera adicionado ao git)
	use o comando "git add ." (o ponto é para mostrar o local a ser feito [ no caso ja esta dentro da pasta do projeto])

7- primeiro comit (commit: grava alteracoes feitas em banco de dados) com a mensagem sendo o comando utilizado
	git commit -m "rails new help_desk -B"
	commit : guarda dados alterados
	-m : mensagem

8- instalando dependencias globalmente
   será gerado o arquivo Gemfile.lock, que trava as versoes das bibliotecas (bloquei atualizacoes das bibliotecas, evitando caso haja atualizacao nova mas que tenha algum bug e pode estragar o projeto)
	bundle install --verbose 
	git ad Gemfile.lock  (criando arquivo que bloqueia atualizacao de bibliotecas)
	git commit -m "bundle install --verbose # Gemfile.lock" (salvar e mostra na msg o comando que usei)

9- rodar o servidor em background, navegar para http://localhost:3000/
	reails server -d  (-d: deixa o server rodando em segundo plano deixando o pront de comando livre pra ser usado)

10- criar um controler com 4 paginas estaicas
	rails g controller Paginas home recurso preco contato
	g: generation
	controller : controlador
11- criar uma rota root em "config/routes.rb: root 'paginas#home'
	abra o sublime e siga os passos ate a aba routes.rb
	faça a alteracao na linha # root ...
	retire o # para deixar de ser um comentario

- depois no pront digite os comandos: 
	git add .
	git commit -m "rails g controller Paginas home recurso preco contato"
	git add . : (especifica o local onde sera feito o processo)
	git commit -m... : guarda as alteracoes e mostra na mensagem o comando feito)

12- baixar o css do bootstrap em "public/bootstrap.min.css"
	wget -O public/bootstrap.min.css 	http://netdna.bootstrapcdn.com/bootstrap/3.1.1/css/bootstrap.min.css
	wget: baixar da internet

13- linkar no "app/views/layouts/application.html.erb: <link rel="stylesheet" href="/bootstrap.min.css">
	no sublime, siga o caminho ate application.html.erb
	adicionar o link " <link rel="stylesheet" href="/bootstrap.min.css"> "
	tem que ser adicionado logo abaixo do titulo
	
	" <title>HelpDesk</title>
  	  <link rel="stylesheet" href="/bootstrap.min.css"> "

13.1- colocar o layout do site dentro de um container no mesmo arquivo.
	
	" <div class="container">
	    
		<%= yield %>

	  </div>

	volte ao pront de comando e execute os comandos: 
	git add .
	git commit -m "add bootsrap.min.css"
